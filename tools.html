The first experince with GITHUB

Well, thanks to GitHub, commit history is being logged and keeps it easy for every member to see which member was the last one to edit the code and what that member edited. So we will have a clear picture of what we have done. And also, commit history made our lives easier. In the event of any member accidentally deleting a wrong line of code, we can check for the change history and fix it in time. And last is commit history, which enables us to know what each member had to do to make the website look like it does at the moment. For Long, he is in charge of the content that Adam will upload to the html.index. Adam is also in charge of the main code for the website. For Lam Anh, his job is to double check the code with Adam and Duy. In the CSS part of the website, each member uses their own knowledge to make the CSS for the website.
